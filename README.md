# Usage
* Add / Replace the code in the snippets / distractor directories.
    * **IMPORTANT**: Snippet & Distractor text file should share the exact same name.
* Run: `python3 parsons.py`

# Considerations
* Only works on MacOs / Linux. Figure out Windows support with AI or something, idc.
* Designed mainly for Java, snippets are from: https://github.com/effective-software-testing/code
* Distractors were generated via an LLM
